# pyLyrics
lyricsplugin.com interface

## Usage
```
  pyLyrics -a <artist> -t <title> [-o file] [-s] [-r] [-v]  

  -h,  --help          Show this message  
  -a,  --artist        Song artist (required)  
  -t,  --title         Song title (required)  
  -o,  --output        Write output to file  
  -s,  --silent        Disable print output to STDOUT
  -r,  --raw           Print lyrics only  
  -v,  --version       Print version information  

  example:  
  lyrics -a "ABBA" -t "Money"  
```
